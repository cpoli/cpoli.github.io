chain_tb¶

The chain_tb package models a one-dimensional tight-binding dimer chain with real and/or complex valued hoppings and onsite energies. Defects can be introduced by locally modifying the on-site energies and/or the hoppings.

chain_tb is available at https://github.com/cpoli/chain_tb

class chain_tb.chain_tb(sites, ta, tb, ea=0j, eb=0j, ext='png')[source]¶

Bases: object

This is the only class of the chain_tb module which contains all information for the one dimensional tight-binding model.

Parameters:
  • sites – Number of sites of the chain (positive integer).
  • ta – Intradimer coupling, connects the A sites to the B sites.
  • tb – Interdimer coupling, connects the B sites to the A sites.
  • ea – Default value 0. Onsite energies of the A sites.
  • eb – Default value 0. Onsite energies of the B sites.
  • ext – Default value ‘png’. Extension of the figures to be saved.
 
# Examples
# Creates a chain monomer with identical hoppings:
ch = chain_tb(sites=22, ta=2., tb=2.)
# Creates a chain with n sites and alternating hoppings:
ch = chain_tb(sites=21, ta=2+0.5j, tb=1-1j, ea=1+1j, eb=1-1j)

set_def_dim(index)[source]¶

Introduce defects in the dimerization pattern by exchanging the intradimer hoppings with the interdimer hoppings starting from the ith site (value(s) containing in index).

Parameters:index – Array containing the indices of the sites.

Note

Dimerization defects start from the 3rd site (index = [2]).

# Examples
ch = chain_tb(sites=11, ta=2., tb=1.)
# chain with 11 sites:
# A==B--A==B--A==B--A==B--A==B--A
# introduce two dimerization defects
ch.set_def_dim([2, 6])
# the first index value 2 changes the hoppings into:
# A==B--A--B==A--B==A--B==A--B==A
# note that the 3rd site is now connected by two weak bonds.
# The second index value 6 changes the hoppings into:
# A==B--A--B==A--B==A==B--A==B--A
# note that the 7th site is now connected by two strong bonds.
set_disorder(strength)[source]¶

Introduce uniform hopping disorder for all the hoppings.

Parameters:strength – disorder strength, hopping are between [t-strength/2: t+strength/2].

Note

In the case of alternating couplings denoted t_min and t_max, the underlying structure of the chain can be broken if strength > t_max-t_min. Weak couplings can then be larger than strong couplings.

# Examples
ch = chain_tb(sites=21, ta=2., tb=1.)
# if
ch.set_disorder(.5)
# disorder does not break the underlying structure of the chain.
ch.set_disorder(1.)
# maximal disorder of the chain without breaking its underlying structure.
ch.set_disorder(2.)
# breaks the underlying structure of the chain.

  • val_pola – is the lower, higher bound when using the larger, smaller selection rule. Note that val_pola .

ch.eig()
# the eigenenergies and eigenstates are required.

get_pumping(psi_init, hop_new, dz=0.05, steps_pump=100, norm='int')[source]¶

Obtain the time evolution of the field submitted to adiabatic modifications of the hopping configuration. The new hopping configurations are contained in the matrix hop_new.

The time evolution is split in three different parts:

  • Stationary time evolution in the initial hopping configuration (the one of the instance of class).
  • Transient time evolution going linearly from the initial hopping configuration to the new one.
  • Stationary time evolution in the new hopping configuration.

Note

The last two parts of the time evolution is repeated times if hop_new contains hopping configurations.

Parameters:
  • psi_init – Initial state.
  • hop_new – List of size (sites-1, N) where N is the number of hoppings configugation.
  • dz – Step in time.
  • steps – Number of steps.
  • steps_pump – Number of steps during the transition.
  • norm – Default value ‘int’. Normalize the intensity at each time step: * ‘no’: Does not renormalize. * ‘int’: renormalize according to . * ‘max:’ renormalize according to .

Note

The normalization, automatically performed in the Hermitian case, does not hold if ea, eb, ta, or tb are complex variables. The normalization can then be useful in presence of amplification/absorption or to plot the time evolution.
